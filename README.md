# Macronutrient Tracker

<a href="https://macronutrients.vercel.app/">Macronutrients</a>

Macronutrient tracking system that calculates and updates daily intake
based on user input and predefined nutritional values.

## Features
- Macronutrient calculation logic (protein, carbs, fats)
- Real-time updates based on user input
- Predefined food list with nutritional values
- Manual food entry support
- Data persistence using localStorage

## Notes
This project focuses on data processing, validation, and rule-based logic,
rather than UI design.
